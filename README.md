# arigo-vscode-app
Prepare Visual Code for Arigo Software GmbH App developing

If you install this package with
```
npm install arigo-software/arigo-vscode-app
```

in your Visual Code workspace.  
The workspace is prepared for App developing.

This includes settings for:
* ESLint
* SFTP Connection to the target
* Schema to edit tree files
